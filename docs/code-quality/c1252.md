# <a name="c1252"></a>C1252
C1252 Uyarı: eklentileri arasında eksik veya döngüsel bağımlılık: GUID gerektirir  
  
 Kod çözümleme aracı değil çözümlenmekte koddaki eklentisi bağımlılıkları olan bir iç hata olduğunda bu uyarı bildirir.
