# <a name="c26167"></a>C26167
C26167 Uyarı: büyük olasılıkla unheld kilidin açılması \<kilit > işlevindeki \<func >.

 Uyarı C26167 benzer uyarı [C26117](../code-quality/c26117.md) anlamlılık düzeyi daha düşük olması dışında. Örneğin, işlevi ek açıklama hataları içerebilir.

## <a name="example"></a>Örnek
 Aşağıdaki kod C26110 yanı sıra C26167 oluşturur.

```cpp

typedef struct _DATA {
    CRITICAL_SECTION cs;
} DATA;

_Releases_lock_(p->cs) void Leave(DATA* p) {
    LeaveCriticalSection(&p->cs); // OK
}
void ReleaseUnheldLock(DATA* p) { // Warning C26167
    int i = 0;
    Leave(p); // Warning C26110
}

```

## <a name="example"></a>Örnek
 Aşağıdaki kod, bu uyarılar düzeltebilirsiniz.

```cpp

typedef struct _DATA {
    CRITICAL_SECTION cs;
} DATA;

_Releases_lock_(p->cs) void Leave(DATA* p) {
    LeaveCriticalSection( &p->cs );
}

void ReleaseUnheldLock( DATA* p ) {
    EnterCriticalSection( &p->cs );
    int i = 0;
    Leave(p);
}

```
