# <a name="c28020"></a>C28020
C28020 Uyarı: ifade \<expr > konumundaki bu çağrısı doğru değil  
  
 Bu uyarı bildirilen zaman _Satisfies\_ listelenen ifadesi geçerli değil. Sık bu yanlış bir parametre belirtir.  
  
 Bu işlev bildirimi üzerinde meydana gelirse, ek açıklamalar mümkün olmayan bir koşulu belirtin.
