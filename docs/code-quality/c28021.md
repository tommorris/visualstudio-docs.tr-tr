# <a name="c28021"></a>C28021
C28021 Uyarı: parametre \<param > ile Açıklama \<Milattan > bir işaretçi olmalıdır  
  
 Açıklama nesne bir işaretçi türü değilse bu uyarı bildirilir. Bu ek açıklama kullanılamaz `void` veya tam sayı türleri.
