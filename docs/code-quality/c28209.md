# <a name="c28209"></a>C28209
C28209 Uyarı: sembol bildirimi çakışan bir bildirime sahip

 Bu uyarı bir hatalı oluşturulmuş ek açıklama bildirimi gösterir. Bu uyarı, normal kullanımda hiçbir zaman gerçekleşeceğini.
