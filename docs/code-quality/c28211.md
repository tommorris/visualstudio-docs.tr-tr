# <a name="c28211"></a>C28211
C28211 Uyarı: statik içerik adı SAL_context için bekleniyor  
  
 Bu uyarı belirten işlenenin `_Static_context_` ek açıklama aracını tanımlı bir içerik adı olmalıdır. Bu uyarı, normal kullanımda hiçbir zaman gerçekleşeceğini.
