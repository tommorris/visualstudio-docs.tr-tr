# <a name="c28212"></a>C28212
C28212 Uyarı: işaretçi ifade eklenti için bekleniyor  
  
 Bu uyarı numaralı parametre açıklamanın (not işlevi açıklama) için bir işaretçi veya dizi türünde olması bekleniyor, ancak bazı diğer türüyle karşılaşıldı gösterir. Ek açıklamanın düzeltilmesi gerekiyor.
