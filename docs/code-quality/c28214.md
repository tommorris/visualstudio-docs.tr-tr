# <a name="c28214"></a>C28214
C28214 Uyarı: öznitelik parametre adları p1... olmalıdır p9

 Bu uyarı bir ek açıklamanın bildirimi oluşturmak, parametre adları... p1 sınırlı olduğunu gösterir p9. Bu uyarı, normal kullanımda hiçbir zaman gerçekleşeceğini.
