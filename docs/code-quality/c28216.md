# <a name="c28216"></a>C28216
C28216 Uyarı: _Check_return\_ ek açıklama yalnızca uygulanır belirli bir işlev parametresi için sonrası koşulları.

 `_Check_return_` Ek açıklama uygulanmış bir bağlamda dışında `__post`; gereksinim duymayabilirsiniz bir `__post` (veya `__drv_out`) değiştiricisi veya yerleştirilebilir yanlış.
