# <a name="c28224"></a>C28224
C28224 Uyarı: ek açıklama parametreleri gerektirir  
  
 Bu uyarı adlı ek açıklama parametresiz kullanıldığında ve en az bir parametre gerekli olduğunu gösterir. Bu standart makroları ek açıklamalar için kullanılıyorsa olası olmamalıdır; Bu uyarı, normal kullanımda görüntülenmesine beklenmiyor.
