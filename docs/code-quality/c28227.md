# <a name="c28227"></a>C28227
Uyarı C28227: Ek açıklama ayrıca bir PrimOp olamaz (önceki bildirimi bakın)

 Bu uyarı adlı ek açıklama sıradan açıklamanın bildirilmiş ve ayrıca daha önce PrimOp bildirildi gösterir. Bu standart makroları ek açıklamalar için kullanılıyorsa olası olmamalıdır; Bu uyarı genel kullanımda görüntülenmesine beklenmiyor.
