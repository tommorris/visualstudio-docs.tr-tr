# <a name="c28228"></a>C28228
C28228 Uyarı: ek açıklama parametresi: türü ek açıklamalar kullanamaz  
  
 Bu uyarı, bir parametre desteklenmeyen türü olduğunu gösterir. Ek açıklamalar yalnızca türleri sınırlı sayıda parametre olarak kullanın. Bu standart makroları ek açıklamalar için kullanılıyorsa olası olmamalıdır; Bu uyarı genel kullanımda görüntülenmesine beklenmiyor.
