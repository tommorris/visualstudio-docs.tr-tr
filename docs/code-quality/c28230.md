# <a name="c28230"></a>C28230
C28230 Uyarı: parametre hiç üye türü.  
  
 Bu uyarı bir ek açıklamanın bağımsız erişmeyi denediği gösterir bir `struct`, `class`, veya `union` ve adlandırılmış üye yok.
