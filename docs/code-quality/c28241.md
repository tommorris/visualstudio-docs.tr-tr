# <a name="c28241"></a>C28241
C28241 Uyarı: parametresindeki işlevi için ek açıklama tanınmıyor  
  
 Tanınmayan ek açıklama adı kullanılır. Bu standart makroları ek açıklamalar için kullanılıyorsa olası olmamalıdır; Bu uyarı genel kullanımda görüntülenmesine beklenmiyor.
