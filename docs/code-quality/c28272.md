# <a name="c28272"></a>C28272
C28272 Uyarı: ek açıklamanın işlevi, incelerken parametresi için işlev bildirimi ile tutarsız.

 Bu uyarı ek açıklamalar, analiz ediliyor kod değil hata gösterir. Bir işlev tanımının görünen ek açıklamalar, bildirim görünmesini olanla tutarsız. İki ek açıklamaları eşleşecek şekilde çözülmesi gerekir.
