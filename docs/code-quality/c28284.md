# <a name="c28284"></a>C28284
C28284 Uyarı: simge için koşulları işlev olmayan simgelerini şu anda desteklenmemektedir  
  
 Bu uyarı gösterir koşullu bir ek açıklama (koşulu, `__drv_when`) dışında bir işlev üzerinde bulunamadı.
