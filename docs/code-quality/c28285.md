# <a name="c28285"></a>C28285
C28285 Uyarı: işlev, parametre sözdizimi hatası  
  
 Kod çözümleme aracı model dosyasında olası hatayla karşılaştığında bu uyarı bildirir. Birkaç kaynak dosya hataları gibi hataları da neden olabilir.
