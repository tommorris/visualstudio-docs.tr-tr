# <a name="c28290"></a>C28290
C28290 Uyarı: işlevi için ek açıklama parametreleri gerçek sayısından daha fazla Externals içerir  
  
 İşlev için ek açıklama parametreleri gerçek sayısından daha fazla Externals içerdiğinde kod analizi aracı bu uyarı bildirir.
