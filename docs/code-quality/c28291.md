# <a name="c28291"></a>C28291
C28291 Uyarı: Post null/notnull adresindeki deref düzeyi 0'dır işlevi için anlamsız \<x > param adresindeki \<numarası >  
  
 Kod çözümleme aracı post koşul bir başvuru düzeyi sıfır parametresi null/boş olmayan özellik belirtildiğinde bu uyarı bildirir. Bu hata düzeyi sıfır değiştiremezsiniz bir değerinde dereference oluşur.
