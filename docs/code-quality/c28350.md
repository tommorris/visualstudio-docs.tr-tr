# <a name="c28350"></a>C28350
C28350 Uyarı: ek açıklamanın \<ek açıklama > koşullu kullanılamaz bir durumda açıklar.

 Genellikle C/C++ türü burada Denetlenmekte olan ek açıklamanın uygulandığında bu uyarı oluşturulur.
