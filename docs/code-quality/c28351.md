# <a name="c28351"></a>C28351
C28351 Uyarı: ek açıklamanın \<ek açıklama > dinamik bir değer (bir değişken) koşulunda burada kullanılamaz açıklar.  
  
 Ek açıklamanın C/C++ türü burada Denetlenmekte olan uygulandığında, bu uyarıyı bildirilir.  
  
 İfade `_When_` bir sabite değerlendirmelisiniz. `_When_` Göz ardı edilir.
