# <a name="c6995"></a>C6995
C6995 Uyarı: XML günlük dosyası kaydedilemedi  
  
 Bu uyarı, kod analizi aracı Kod Analizi çıktısı hatasını günlük oluşturulamıyor gösterir.  
  
 Bu hata bir disk hatasından kaynaklanıyor olabilir veya belirtilen dizindeki bir dosya oluşturma izni olmadığını gösterir.
