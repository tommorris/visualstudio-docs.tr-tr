# <a name="ca0051"></a>CA0051
Hiçbir kural seçilmedi.

Bu hata, yönetilen kod çözümleme aracı etkindir, ancak hiçbir kural seçilen gösterir. Bu hatayı düzeltmek için uygun kuralları seçin.
