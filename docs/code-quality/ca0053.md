# <a name="ca0053"></a>CA0053
Kural derlemesi yüklenemedi.

 Bu, bir kural derlemesi tarafından yansıma yüklenemedi gösterir. Kullanıcının doğru şekilde oluşturulmuş olmayan özel kurallar varsa bu gerçekleşebilir veya gibi dosya sistemi erişimi reddedildi çözümleme aracı diğer nedenlerle derlemesi yüklenemiyor.
