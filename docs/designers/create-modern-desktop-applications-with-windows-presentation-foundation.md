# <a name="create-modern-desktop-applications-with-windows-presentation-foundation"></a>Windows Presentation Foundation ile modern masaüstü uygulamaları oluşturma

Visual Studio'da Windows Presentation Foundation (WPF), modern iş kolu satır Windows Masaüstü uygulamaları oluşturmak için birleşik bir programlama modeli geliştiricilere sağlar.  

Ayrıntılı izlenecek yollar için ile başlayan [WPF ile çalışmaya başlama](../designers/getting-started-with-wpf.md) ve onun ikincil konuları izleyin.

Kullanılabilen araçlar karşılaştırması için bkz: [Visual Studio ve Visual Studio için Blend'de XAML tasarlama](../designers/designing-xaml-in-visual-studio.md).
