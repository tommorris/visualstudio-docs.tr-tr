# <a name="color-value-reference-for-visual-studio"></a>Visual Studio için renk değeri başvurusu
Bu konu, yeni renk belirteçleri oluşturmak için bir başvurudur. Yeni kullanıcı arabirimini oluştururken, her ana Tema renkleri ve yüksek karşıtlıklı tema için bu öğe için sistem renk adı seçmek için mevcut UI benzer renkleri kullanabilirsiniz. 

Yeni renk belirteçleri oluşturma hakkında daha fazla uygulama bilgisi için bkz: [renkli Tema oluşturma araçları](../../extensibility/internals/color-theming-tools.md).

Renk belirteçleri varolan kullanıcı Arabirimi ile kullanmak, bkz: [renkleri Visual Studio için paylaşılan](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

Bu başvuru kılavuzunda kullanırken unutmayın:

-   **Her zaman ön ve arka plan tonlar birlikte kopyalayın.** Tonları olduğu gibi çünkü bir ön plan bir kullanıcı Arabirimi öğesi ve başka bir arka plan seçmeyin.  

-   **Bu onaltılık olmayan renk belirteçleri kodlarıdır.** Belirteçleri nasıl gelecekte değişebilir denetleyemeyeceğiniz için bu kullanıcı Arabirimi öğeleri gerçek renk belirteci adlarını kullanamazsınız. Bunun yerine, kendi kategoride belirteçleri oluşturun ve tabloda gösterilen ton değerleri kullanın. Varolan UI bulunabilir için belirteçleri renk [renkleri Visual Studio için paylaşılan](../../extensibility/ux-guidelines/shared-colors-for-visual-studio.md).

- **Yüksek Karşıtlık örnekleri yalnızca karşılaştırma için gösterilir.** Yüksek Karşıtlık UI için sistem renk adını kullanın ve bu örnekleri üzerinde Damlalık veya Renk Seçici kullanmayın. Bu konudaki yüksek karşıtlık örnekleri Windows yüksek karşıtlık #1 temada yansıtır. Renkleri diğer yüksek karşıtlıklı tema farklı görünecektir. 

-   **En iyi sonuçlar için kullanıcı Arabirimi ile örnektekine benzer çifti rengini belirler.**
