# <a name="c28300"></a>C28300
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

Bu konuda en son sürümünü şu yolda bulunabilir: [C28300](https://docs.microsoft.com/visualstudio/code-quality/c28300).  
  
Uyarı C28300: < parameter_name >: < operator_name > işleci için uyumsuz olan türlerde ifade işlenenleri  
  
 Bu uyarı, bir parametre işlenen türlerinde işleci ile uyumlu olmadığında bildirilir.
