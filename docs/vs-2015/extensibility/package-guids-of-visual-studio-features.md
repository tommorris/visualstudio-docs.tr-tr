 Bazı özellikler tamamen kaldırılamadı. Örneğin, kaldırmak için kaydı hiçbir paket var. **sınıf görünümü** ve onun ilişkili menü seçenekleri ve Hizmetleri. **Sınıf görünümü** penceresi Visual Studio ortam diğer temel IDE özellikleri sağlayan paket tarafından sağlanır. Kaldırmak istiyorsanız **sınıf görünümü**, kaldırma gerekirdi **Bul ve Değiştir**, ortam **seçenekleri** sayfaları **komut penceresi**ve **çıkış** penceresi.
