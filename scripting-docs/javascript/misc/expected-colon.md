# <a name="expected-3939"></a>Beklenen &#39;: &#39;
Üçlü koşullu işleç kullanan bir ifade oluşturmayı denedi, ancak iki nokta üst üste ikinci ve üçüncü işlenenleri arasındaki içermiyordu. İkinci (true) ve üçüncü (false) arasında iki nokta üst üste Üçlü (üç işlenen) koşullu işleç gerektirir işlenen.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   İkinci ve üçüncü işlenenler arasında iki nokta üst üste koyun.
