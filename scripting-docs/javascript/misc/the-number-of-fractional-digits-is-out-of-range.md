# <a name="the-number-of-fractional-digits-is-out-of-range"></a>Kesir basamakları sayısı aralık dışı
İşleve geçersiz bir bağımsız değişken geçirme girişiminde **Number.prototype.toExponential**. İşlevin bağımsız değişkeni **toExponential()** 0 ile 20 (dahil) arasında olmalıdır.  
  
### <a name="to-correct-this-error"></a>Bu hatayı düzeltmek için  
  
-   Bağımsız değişkeni olun **toExponential()** çok büyük veya çok küçük değil.
