# <a name="assignment-operator--javascript"></a>Atama İşleci (=) (JavaScript)
Bir değeri bir değişkene atar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
result = expression  
```  
  
## <a name="parameters"></a>Parametreler  
 `result`  
 Herhangi bir değişken.  
  
 `expression`  
 Herhangi bir sayısal ifade.  
  
## <a name="remarks"></a>Açıklamalar  
 = İşleci onu içeren ifadeleri bir değere sahip olması için diğer işleçleri gibi davranır. Atama İşleçleri gibi zincir, yani: `j = k = l = 0`. Bu durumda `j`, `k`, ve `l` sıfıra eşit.
