# <a name="debugdebuggerenabled-property"></a>Debug.debuggerEnabled Özelliği
Hata ayıklama betik bağlam için etkin olup olmadığını belirler. Hata ayıklama etkin veya olabilir bir hata ayıklayıcısı ekli olup olmadığına bakılmaksızın devre dışı.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
var dbgEnabled = Debug.debuggerEnabled;  
```
