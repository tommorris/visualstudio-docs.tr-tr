# <a name="division-assignment-operator---javascript"></a>Bölme atama işleci (-) (JavaScript)
Bir ifadenin değerini bir değişken değerle böler ve sonucun değişkenine atar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
result /= expression   
```  
  
## <a name="parameters"></a>Parametreler  
 `result`  
 Herhangi bir sayısal değişkeni.  
  
 `expression`  
 Herhangi bir sayısal ifade.  
  
## <a name="remarks"></a>Açıklamalar  
 Kullanarak `/=` işleci aynıdır belirtme: `result = result / expression`.
