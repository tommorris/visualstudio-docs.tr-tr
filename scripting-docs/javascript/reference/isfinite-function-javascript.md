# <a name="isfinite-function-javascript"></a>isFinite İşlevi (JavaScript)
Sağlanan sayıyı sınırlı olup olmadığını belirler.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
isFinite(number)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli `number` bağımsız değişkeni olan herhangi bir sayısal değer.  
  
 `isFinite` İşlev döndürür `true` varsa `number` dışındaki herhangi bir değer olan `NaN`, negatif sonsuz ya da pozitif sonsuzluk. Bu üç durumda döndürdüğü **false**.
