# <a name="mathexp-function-javascript"></a>Math.exp İşlevi (JavaScript)
Üssü e (Doğal logaritmanın tabanı) döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.exp(number)   
```  
  
## <a name="remarks"></a>Açıklamalar  
 Gerekli `number` bağımsız değişkeni gücünü temsil eden bir sayısal ifadedir *e*.  
  
 Dönüş değeri olan bir sayıdır. Sabit *e* Euler number, 2.71828 yaklaşık eşit olduğundan ve `number` sağlanan bağımsız değişken.
