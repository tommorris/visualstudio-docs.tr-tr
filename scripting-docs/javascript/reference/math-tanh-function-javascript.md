# <a name="mathtanh-function-javascript"></a>Math.tanh İşlevi (JavaScript)
Bir sayının hiperbolik tanjantını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
Math.tanh(  
x  
)  
  
```  
  
## <a name="remarks"></a>Açıklamalar  
 Dönüş değeri olarak aynıdır `(exp(x) - exp(-x))/(exp(x) + exp(-x))`. Varsa `x` olan `NaN`, sonuç `NaN`.
