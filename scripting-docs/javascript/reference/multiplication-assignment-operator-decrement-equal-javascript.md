# <a name="multiplication-assignment-operator--javascript"></a>Çarpma Atama İşleci (*=) (JavaScript)
Bir değişkenin bir ifadenin değerini değerle çarpar ve sonucu değişkenine atar.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
  
result *= expression  
```  
  
## <a name="parameters"></a>Parametreler  
 *Sonuç*  
 Herhangi bir değişken.  
  
 *ifade*  
 Herhangi bir ifade.  
  
## <a name="remarks"></a>Açıklamalar  
 Kullanarak  **\* =**  işleci tam olarak aynı olup belirtme:  
  
```JavaScript  
result = result * expression  
```
