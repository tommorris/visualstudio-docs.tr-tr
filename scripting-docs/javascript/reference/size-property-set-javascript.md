# <a name="size-property-set-javascript"></a>size Özelliği (Küme) (JavaScript)
Bir kümedeki öğe sayısını döndürür.  
  
## <a name="syntax"></a>Sözdizimi  
  
```JavaScript  
sizeVar = setObj.size  
```  
  
#### <a name="parameters"></a>Parametreler  
 `sizeVar`  
 Gerekli. Herhangi bir sayı.  
  
 `setObj`  
 Gerekli. Tüm `Set` nesnesi.
