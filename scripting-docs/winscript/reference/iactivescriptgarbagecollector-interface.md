# <a name="iactivescriptgarbagecollector-interface"></a>IActiveScriptGarbageCollector Arabirimi
Çöp toplama başlatmak için bir yöntem sağlar. Bu arabirim, kullanıcıların kaynakları temizlemek istediğiniz etkin komut dosyası motorları tarafından uygulanmalıdır.  
  
## <a name="methods"></a>Yöntemler  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[IActiveScriptGarbageCollector::CollectGarbage](../../winscript/reference/iactivescriptgarbagecollector-collectgarbage.md)|Çöp toplama başlatır.|
