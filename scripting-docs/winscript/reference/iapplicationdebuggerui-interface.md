# <a name="iapplicationdebuggerui-interface"></a>IApplicationDebuggerUI Arabirimi
Hata ayıklayıcı tümleşik geliştirme ortamı (IDE) uygulanan (ek olarak `IApplicationDebugger`) dış bileşen hata ayıklayıcı kullanıcı arabirimini (UI) üzerinde daha fazla denetim sağlamak için.  
  
 Kaynağından devralındı yöntemleri yanı sıra `IUnknown`, `IApplicationDebuggerUI` arabirimi aşağıdaki yöntemleri sunar.  
  
## <a name="methods-in-vtable-order"></a>Vtable sırayla yöntemleri  
  
|Yöntem|Açıklama|  
|------------|-----------------|  
|[IApplicationDebuggerUI::BringDocumentToTop](../../winscript/reference/iapplicationdebuggerui-bringdocumenttotop.md)|Kullanıcı arabirimi hata ayıklayıcı üst belirtilen hata ayıklama belgeye içeren pencere getirir.|  
|[IApplicationDebuggerUI::BringDocumentContextToTop](../../winscript/reference/iapplicationdebuggerui-bringdocumentcontexttotop.md)|Hata ayıklayıcı kullanıcı arabiriminde üst verilen belgedeki bağlamına içeren pencere getirir ve bağlam penceresine kaydırır.|
