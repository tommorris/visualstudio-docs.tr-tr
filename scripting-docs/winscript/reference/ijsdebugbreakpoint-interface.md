# <a name="ijsdebugbreakpoint-interface"></a>IJsDebugBreakPoint Arabirimi
Bir kesme noktası temsil eder.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
IJsDebugBreakPoint : public IUnknown;  
```  
  
## <a name="members"></a>Üyeler  
  
### <a name="public-methods"></a>Ortak Yöntemler  
  
|Ad|Açıklama|  
|----------|-----------------|  
|[Ijsdebugbreakpoint::Delete yöntemi](../../winscript/reference/ijsdebugbreakpoint-delete-method.md)|Kesme noktası siler.|  
|[Ijsdebugbreakpoint::disable yöntemi](../../winscript/reference/ijsdebugbreakpoint-disable-method.md)|Kesme noktası devre dışı bırakır.|  
|[Ijsdebugbreakpoint::enable yöntemi](../../winscript/reference/ijsdebugbreakpoint-enable-method.md)|Kesme noktası sağlar.|  
|[Ijsdebugbreakpoint::getdocumentposition yöntemi](../../winscript/reference/ijsdebugbreakpoint-getdocumentposition-method.md)|Kesme noktası burada bağlıydı deyim konumunu döndürür.|  
|[Ijsdebugbreakpoint::IsEnabled yöntemi](../../winscript/reference/ijsdebugbreakpoint-isenabled-method.md)|Kesme noktası etkin olup olmadığını belirler.|
