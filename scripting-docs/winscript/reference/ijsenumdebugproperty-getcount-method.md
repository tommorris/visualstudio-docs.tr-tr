# <a name="ijsenumdebugpropertygetcount-method"></a>IJsEnumDebugProperty::GetCount Metodu
Nesnenin özelliklerini sayısını alır.  
  
## <a name="syntax"></a>Sözdizimi  
  
```  
HRESULT GetCount(  
   ULONG *pCount  
);  
```  
  
#### <a name="parameters"></a>Parametreler  
 `pCount`  
 [out] Özellikleri sayısı.  
  
## <a name="return-value"></a>Dönüş Değeri
