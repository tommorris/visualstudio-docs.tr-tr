# <a name="scriptepropagate-error-code"></a>SCRIPT_E_PROPAGATE Hata Kodu
Bir komut dosyası hatası farklı bir iş parçacığı olabilir çağırana yayılır. Ana bilgisayar hata kodu çağırana geçirmesini.  
  
## <a name="syntax"></a>Sözdizimi  
  
```cpp  
SCRIPT_E_PROPAGATE  0x80020102L  
```
